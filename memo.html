任意のフレーム間表示する

今回はJavaScriptで説明。ゲーム製作で"ここで敵を10フレームだけ表示したい"などのようなときはジェネレータを使うと便利。ジェネータとは、以下のものである。 

	function* generator(){
	    let a = 1;

	    yield a;
	    a++;
	    yield a;
	    a++;
	    yield a;
	}

	let g = generator();

	consol.log(g.next());
	consol.log(g.next());
	consol.log(g.next());
	            
	1
	2
	3

ここでは、ジェネレータの説明はしない。これを使い、例えば 

	function* Generator(frame){
	    for(let i = 0; i < frame; i++){
	        描画関数();
	    }
	}

	毎フレーム呼び出される関数(){
	    g.next();
	}

とすることで、好きな時に「g = Generator(10);」とすれば、その時から10フレームだけ描画することができる。(※変数gは、何も処理しないジェネレータ関数か「Generator(0)」で初期化しておかなければ、エラーが出る。)また、 

	function StartGen(frame){
	    g = Generator(frame);
	}

としておけば、「StartGen(10);」のように普通の関数のように使うことができる。

リスト末尾からのアクセス(python)

pythonでのリスト末尾からのアクセスは 

a = [ 1, 2, 3, 4, 5 ]
print( a[-1] )	# 5
print( a[-2] )	# 4
print( a[-3] )	# 3

であるが、ビット演算子「~」を使えばよりわかりやすくアクセスできる。 

print( ~0 )	# -1
print( ~1 )	# -2
print( ~2 )	# -3


a = [ 1, 2, 3, 4, 5 ]

print( a[~0] )	# 5
print( a[~1] )	# 4
print( a[~2] )	# 3

a[~n]とすると末尾からn番目の要素へアクセスでき、先頭からアクセスするときと同じように書くことができる。わかりやすい。
